# 長照新案訪視紀錄系統 📝

[![線上體驗](https://img.shields.io/badge/立即體驗-Live_Demo-blue?style=for-the-badge&logo=github)](https://71658-dev.github.io/Long-term_Care_Visit_Report/)

專為長照專員訪視人員量身打造的「新案訪視紀錄」輔助工具！
透過簡單直覺的網頁操作，只需點選勾選與填寫關鍵資訊，系統便會為您自動生成排版工整、符合標準格式的訪視報告。大幅減少重複打字與排版的時間，讓您能更專注於個案服務。

👉 **[點擊此處立即體驗系統](https://71658-dev.github.io/Long-term_Care_Visit_Report/)**

---

## ✨ 系統特色

*   **免安裝、隨開即用**：不需要安裝任何軟體或應用程式，只要有網路並使用瀏覽器開啟網頁即可使用。
*   **資料自動暫存**：系統會自動將您輸入的內容暫存在瀏覽器中。即使不小心重整網頁或關閉分頁，下次打開時資料依然存在，不怕心血白費！
*   **所見即所得**：採用「左邊填寫表單，右邊預覽報告」的雙視窗設計。填寫資料的當下，右側的報告畫面會即時更新，隨時確認最終的呈現效果。
*   **自訂選項功能 (選項編輯器)**：內建彈性的選項自訂功能，可根據您實際的訪視需求，靈活新增或調整表單內的選項內容，讓系統更貼近您的工作習慣。

## 🛠️ 核心功能

*   **快速選單輸入**：提供豐富的按鈕與選單，涵蓋基本資訊、個案摘要、醫療與疾病、日常活動、照顧計畫等各個面向，點擊即可快速帶入標準化的專業敘述。
*   **一鍵匯出 Word 檔**：完成報告後，只需按一下按鈕，即可將報告下載為 Word 文件 (`.doc`)。系統已為您預設好「標楷體」與「標準邊界 (1.27cm)」，方便後續直接開啟微調或存檔歸檔。
*   **完美列印與 PDF 匯出**：支援直接列印或匯出為 A4 大小的 PDF 檔。在列印模式下會自動隱藏操作介面與按鈕，只保留乾淨、專業的報告排版。

## 🚀 操作方式

1.  **開啟系統**：點擊上方的 [線上體驗連結](https://71658-dev.github.io/Long-term_Care_Visit_Report/) 進入系統。
2.  **填寫訪視資料**：在畫面左側的各個分類區塊中，依序填寫個案資訊或點選符合現況的選項。
3.  **確認報告內容**：隨時查看右側的「報告預覽區」，確認自動生成的文字敘述是否正確且通順。（右側報告區可獨立上下滑動檢查）
4.  **輸出或列印**：
    *   **下載 Word**：點擊右上角的「匯出 Word」按鈕，即可取得可編輯的文件檔案（檔名會自動帶入今日日期）。
    *   **列印 / 另存 PDF**：點擊「列印 / 儲存為 PDF」按鈕，即可利用瀏覽器的列印功能直接印出實體文件，或在印表機選項中選擇「另存為 PDF」。
